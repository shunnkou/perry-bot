## Features

Commandline dashboard
~ View your upcoming habits.
~ Stay motivated by tracker streaks.

Trackers
~ Water tracker.
~ Mood tracker.
~ Habit tracker.

Visualize your tracked data
~ With `matplotlib`.

Local data storage
~ With a SQLite database.
